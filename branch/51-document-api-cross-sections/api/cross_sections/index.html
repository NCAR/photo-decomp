Cross Sections#

A cross_section_t, or one of its subclasses, calculates a chemical cross section for one or more chemical reactions.

The cross sections used by tuv-x are specified by the configuration. See Cross Sections for more information. Each cross section is built by the tuvx_cross_section_factory and stored in the cross_section_warehouse_t.
